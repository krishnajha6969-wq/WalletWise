# 💰 WalletWise — Financial Tracker Web App

## 📌 Project Description

WalletWise is a full-stack financial tracking web application that allows users to manage their income and expenses efficiently.

The system helps users track spending, categorize transactions, and maintain better financial discipline using a modern and responsive interface.

---

## 🎯 Objectives

* To build a real-time expense tracking system
* To implement secure user authentication
* To provide structured financial data using categories
* To understand full-stack development and deployment

---

## ⚙️ Features

* User Registration & Login (JWT Authentication)
* Add / Edit / Delete Transactions
* Categorized Expense Management
* Default Categories (Auto-seeded)
* Secure Backend APIs
* Responsive UI

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Axios

### Backend

* Node.js
* Express.js
* Sequelize ORM

### Database

* PostgreSQL (Neon Cloud Database)

### Deployment

* Frontend: Vercel
* Backend: Render
* Database: Neon

---

Backend API (Node.js + Express)

## 🔐 Security Implementation

* Password hashing using bcrypt
* JWT-based authentication
* Protected API routes
* Environment variables for sensitive data

---

## 🚀 Setup Instructions

### Backend Setup

```bash
cd server
npm install
npm run dev
```

---

### Frontend Setup

```bash
cd client
npm install
npm start
```

---

### Environment Variables (.env)

```env
DATABASE_URL=your_database_url
JWT_SECRET=your_secret
JWT_EXPIRES_IN=7d
PORT=5000
```
